Select Tasks for Portfolio

Select the tasks you want to include in your portfolio. Make sure that you select enough tasks you include demonstrate that you have met all of the unit's learning outcomes. Doubtfire will then incorporate your work from these tasks into your portfolio.

Submit Tasks to Include

Please submit tasks to Doubtfire to include in your portfolio. You need to make sure that the tasks you include will demonstrate all of the unit's learning outcomes. Processing time may take a few minutes, so check back if you have recently submitted work.

Tasks with Changes

If you have significantly change a task, revisit the tasks and you can upload the files again. This wont change the state of the task, but will allow you to upload files that include new changes.
